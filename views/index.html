Request Header Parser API

A simple API that implements the following user story:

  • I can get the IP address, language and operating system for my browser.

Example Usage:

GET /api/whoami
https://strychemi-request-header-api.glitch.me/api/whoami

Results in the following JSON response:

      {
        "ipAddress": "108.28.102.99,::ffff:10.10.11.164,::ffff:10.10.10.252",
        "language": "en-US",
        "browserOS": "(Macintosh; Intel Mac OS X 10_13_3)"
      }
